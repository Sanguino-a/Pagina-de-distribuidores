# Git Commands to Push Changes to Repository

## Step 1: Check current status
git status

## Step 2: Add all changes
git add .

## Step 3: Commit with descriptive message
git commit -m "Add Firebase hosting configuration and update .gitignore"

## Step 4: Push to remote repository
git push origin main

# Alternative commands (if your main branch is called 'master'):
# git push origin master

## Optional: Check what will be pushed
git log --oneline -5
